Memory Game

Instrucciones

Bienvenido/a, dispones de X segundos para encontrar la pareja de todas los colores. Para ello deberás memorizar la posición de cada elemento (color) y la de su pareja. El juego finaliza cuando se cometen X errores, finaliza el tiempo o bien superas con éxito el nivel.

Dispones de varios modos de dificulta teniendo diferente número de errores posibles para cada uno de ellas.

  • Principiante: 2 errores
  • Intermedio: 5 errores
  • Avanzado: 8 errores
